# 文本问答插件 (Text Q&A)

## 插件信息
- 名称: Text Parser
- 版本: 1.0.0
- 作者: Xiaoyi

## 功能简介
这是一个简单的问答学习系统，可以让机器人学习问题和答案，并能够回答已学习的内容。支持添加新问答、删除已有问答，以及自动回复功能。

## 核心功能
1. 学习新的问答
2. 删除已有问答
3. 自动回答已学习的问题
4. 数据持久化存储

## 使用方法

### 学习新问答
格式:
```
学习 问题 答案
```
示例:
```
学习 你好 你好啊
```

### 删除问答
格式:
```
忘记 问题
```
示例:
```
忘记 你好
```

### 提问
直接发送问题即可获得回答:
```
你好
```

## 数据存储
- 文件名: `qa_data.json`
- 格式: JSON
- 编码: UTF-8

### 数据格式示例
```json
{
  "你好": "你好啊",
  "天气": "今天天气不错",
}
```

## 技术实现

### 主要函数
- `load_qa_data()`: 加载问答数据
- `save_qa_data()`: 保存问答数据
- `on_message()`: 消息处理函数

### 依赖项
- json
- os
- asyncio

## 错误处理
1. 学习格式错误提示
2. 删除不存在问题的提示
3. 文件读写异常处理
4. 编码保护

## 注意事项
1. 问答数据保存在同目录下的 qa_data.json 文件中
2. 确保程序对该目录有读写权限
3. 学习新问答时注意使用正确的格式
4. 问题是完全匹配的，需要严格一致才能触发回答

## 示例会话

### 学习示例
```
用户: 学习 今天天气如何 今天天气晴朗
机器人: 我学会了! 问题: 今天天气如何, 答案: 今天天气晴朗

用户: 今天天气如何
机器人: 今天天气晴朗
```

### 删除示例
```
用户: 忘记 今天天气如何
机器人: 我已经忘记了关于'今天天气如何'的回答
```

## 性能考虑
- 数据使用内存字典存储，读写快速
- 每次修改都会同步到文件，保证数据安全
- JSON格式便于人工编辑和查看
